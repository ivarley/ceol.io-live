My Tunes Sorting Test

Test Instructions

  1. Navigate to /my-tunes
  2. Look for the tune count display (e.g., "10 tunes")
  3. Next to it, you should see a button with "a-z"
  4. Click the button to cycle through sort modes:
    • a-z - Alphabetical ascending
    • z-a - Alphabetical descending
    • # - Popularity (highest first), tune type shows tunebook count
    • #ꜜ - Popularity (lowest first), tune type shows tunebook count
  5. Verify that:
    • Tunes are sorted correctly in each mode
    • When in # or #ꜜ mode, the tune type badge shows the tunebook count number
    • When in a-z or z-a mode, the tune type badge shows the tune type (e.g., "reel", "jig")
    • The sort mode persists in the URL (e.g., ?sort=popularity-desc)

Expected Behavior

  • Clicking the sort button cycles through 4 modes in order
  • The icon changes to show current mode: a-z → z-a → # → #ꜜ → (back to a-z)
  • Tunes re-render immediately when sort mode changes
  • The tune type badge switches between showing tune type and tunebook count
  • Sort preference is saved in URL and restored on page reload
