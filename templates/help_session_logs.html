Session Logs

Session logs are records of what happened at each individual session instance. They capture the tunes that were played, organized by sets.

Creating a Log

To create a new log:

  1. Go to your session's page
  2. Click the Add button
  3. Select the date (it defaults to the next scheduled session based on your recurrence)
  4. The new session instance is created with an empty tune list

Adding Tunes to a Log

The tune editor works like a word processor:

  • Type tune names - Press Enter after each tune name, and the system will try to match it to a tune in the database
  • Paste lists - Copy tune names from elsewhere and paste them in; each line becomes a tune
  • Sets - Tunes are grouped into sets. Press Enter twice to start a new set
  • Auto-linking - The system automatically matches tune names to the database and creates links

Editing Tunes

Right-click (or long-press on mobile) any tune to access options:

  • Edit Name - Change how the tune name is displayed
  • Relink - Choose a different tune from the database if the auto-match was wrong
  • Move - Relocate the tune to a different position or set
  • Delete - Remove the tune from the log
  • Info - View tune details and link to TheSession.org

Drag and Drop

You can reorganize tunes by dragging them:

  • Drag within a set to reorder
  • Drag between sets to move a tune
  • The order is saved automatically

Auto-Save

Changes to the tune list are saved automatically. You can also manually save using the Save button. The auto-save interval is configurable in the session settings.

Viewing Historical Logs

The Logs tab shows all session instances, organized by year. Click on any date to view the tunes that were played that night.
