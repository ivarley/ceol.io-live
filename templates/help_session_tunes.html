Session Tunes

Each session has a Tunes tab that shows all the tunes that have been played at that session across all logs. This gives you a comprehensive view of the session's repertoire.

Viewing Session Tunes

The tunes tab displays:

  • Tune Name - The name of the tune, linked to its details on TheSession.org
  • Tune Type - Reel, jig, hornpipe, etc.
  • Play Count - How many times this tune has been logged at this session
  • Last Played - When the tune was most recently logged

Sorting and Filtering

You can organize the tune list by:

  • Popularity - Sort by how often each tune is played
  • Alphabetical - Sort by tune name
  • Type Filter - Show only specific tune types
  • Search - Find specific tunes by name

Tune Details

Click on any tune to see its details:

  • Links to each session instance where the tune was played
  • The tune's page on TheSession.org
  • Sheet music notation (incipit) if available
  • ABC Tools link - Click the ABC icon to open the tune in Michael Eskin's ABC Tools, where you can hear it played back
  • Option to add the tune to your personal tune list

Popular Tunes

The session home page shows the most popular tunes for that session. These are the "core repertoire" - tunes that get played frequently and are good ones to learn if you want to participate more actively.
